☁️ Cấu hình Giai đoạn 1 (Cloud AWS)
Compute
Amazon EC2 (Dòng T3 / M5 instances) — Hỗ trợ Auto-scaling scale theo tải hệ thống.
Database
Amazon RDS (PostgreSQL / MySQL Multi-AZ) — Đảm bảo DB an toàn, tự động backup.
Storage
Amazon S3 — Lưu trữ bucket không giới hạn dung lượng File, Hình ảnh công trường.
Network
AWS VPC + WAF + Application Load Balancer — Bảo mật, phân luồng truy cập nội bộ và bên ngoài.

Máy chủ (x3)
3 Node H3C UniServer R4700 G6 (1U)
CPU: 2x Intel Xeon Gold 6526Y (Tổng 32 Core vật lý/node)
RAM: 256GB ECC DDR5/node
Nguồn: Dual PSU 1300W dự phòng
Ảo hóa
VMware vSphere Foundation (96 Core) — Chạy HA Cluster (High Availability tự động migrate VM).
Lưu trữ
NAS Synology RS2423+
Khả dụng ~12TB HDD (RAID 5) / Phục vụ Database & Lưu trữ.
Mạng & BM
Firewall FortiGate 60F (x2 Active-Passive HA)
Core Switch Maipu NSS5812 (x2 Stackable L3).
⭐ LỘ TRÌNH THỰC HIỆN ĐỀ XUẤT CHO BAN LÃNH ĐẠO
1. Giai đoạn khởi nghiệp (Hiện tại - Hết 2026): Dùng thuần Cloud AWS — Tiêu tốn OPEX linh hoạt để giải quyết ngay bài toán tái khởi động CMS và HRM mà không phải đợi mua sắm thiết bị.
2. Giai đoạn Build in-house (Năm 2027): Tiến hành sắm Server — Đặt mua cụm Dell PowerEdge và xin báo giá chốt Hợp đồng thuê không gian Rack tại các Datacenter chuẩn Tier 3 ở Hà Nội.
3. Giai đoạn Độc lập (Cuối 2027): Migration toàn diện — Rút dần khỏi AWS để tránh phình chi phí đường dài, dịch chuyển dữ liệu về Server của Trung Chính đặt tại DC để tối ưu OPEX trọn đời với cả 14 hệ thống.
