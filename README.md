# 03_WEB_PRESENTATION - MÔI TRƯỜNG TRÌNH DIỄN (THE PRESENTATION LAYER)

Thư mục này đóng vai trò xuất bản giao diện trực quan cho toàn bộ dự án, biến các tài liệu thiết kế khô khan thành Slide/Dashboard chuyên nghiệp, tương tác được.

## Tổ chức Portal (index.html)
Toàn bộ các bản Presentation sẽ được Neo (Endpoint) vào một trang chủ duy nhất là `index.html`. 
- **Cách dùng:** Chỉ cần click đúp mở `index.html` lên bằng trình duyệt, một giao diện Dashboard sẽ hiện ra chứa các Card điều hướng tới các tài liệu trình chiếu khác. Không cần phải mò mẫm thư mục để tìm file mở nữa.

## Lưu trữ Presentation
Tất cả các bản thuyết trình HTML mới cũng sẽ được lưu trực tiếp dưới thư mục này (ví dụ: `CCTC_TC_PRS.html` hiện có).
